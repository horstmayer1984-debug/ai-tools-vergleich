Anthropic hat zwischen Januar und Maerz 2026 22 bedeutende Updates fuer Claude veroeffentlicht. Das sind im Schnitt fast acht pro Monat. Ein viraler X Post von TawohAwa fasst die Entwicklung zusammen. Hier der komplette Ueberblick ueber alle Features und was sie fuer Nutzer bedeuten.

## Alle 22 Updates chronologisch

### Januar 2026

Claude Cowork Launch. Die Desktop App die Claude zum Arbeitsassistenten macht.

### Februar 2026

Opus 4.6 Release mit verbessertem Reasoning und 1 Million Token Kontext in der Beta. Sonnet 4.6 Release als neues Standard Modell. Cowork auf Windows und macOS. PowerPoint Integration fuer direkte Zusammenarbeit in Praesentationen. Excel Integration fuer automatisierte Tabellen und Analysen. Cowork Plugins fuer erweiterbare Skills. Claude Code Security mit Schwachstellen Scans. Claude Code Remote Control. Scheduled Tasks in Cowork fuer wiederkehrende Jobs. Connector im Free Plan fuer alle Nutzer kostenlos.

### Maerz 2026

Claude Memory jetzt kostenlos fuer Free User. Claude Marketplace fuer fertige Skills und Vorlagen. Claude.com Ambassadors Community Programm. Code Review fuer Claude Code. Claude Skills fuer Excel und Slides. Charts und Diagramme direkt im Chat. 1 Million Token Kontextfenster. Dispatch fuer intelligente Task Verteilung. Claude Code Channels fuer Team Entwicklung. Cowork Projects fuer Projektverwaltung. Claude Computer Use fuer Desktop Steuerung. Auto Mode in Claude Code: Claude trifft eigenstaendig Permission Entscheidungen mit integrierten Safeguards.

## Was die wichtigsten Updates in der Praxis bedeuten

### Cowork und Desktop Integration

Claude ist nicht mehr nur ein Chatbot. Cowork macht daraus einen Arbeitsassistenten der mit Excel, PowerPoint, Slack und anderen Tools verbunden ist. Teams die mit Microsoft Office arbeiten koennen Claude direkt in ihren bestehenden Workflow einbinden.

### Opus 4.6 und das 1 Million Token Fenster

Das 1 Million Token Kontextfenster ermoeglicht es komplette Codebasen oder umfangreiche Dokumentensammlungen in einer einzigen Anfrage zu verarbeiten. Fuer Entwickler die grosse Projekte analysieren muessen ist das ein konkreter Produktivitaetsgewinn.

### Auto Mode in Claude Code

Der am 24. Maerz vorgestellte Auto Mode ist das juengste und weitreichendste Update. Claude trifft eigenstaendig Entscheidungen ueber Dateiaenderungen und Terminal Befehle. Sicherheits Checks laufen im Hintergrund. Entwickler muessen nicht mehr jede Aktion einzeln bestaetigen.

### Computer Use und Dispatch

Claude kann jetzt den Mac Desktop steuern. Apps oeffnen, im Browser navigieren, Tabellen ausfuellen. Dispatch ermoeglicht es Aufgaben vom Smartphone aus zu starten. Du weist Claude eine Aufgabe zu und kommst spaeter zu fertiger Arbeit zurueck.

## Was kostenlos und was kostenpflichtig ist

Memory und Connector sind seit Maerz 2026 im Free Plan enthalten. Cowork, Computer Use und Auto Mode erfordern ein Pro oder Max Abo. Pro kostet 20 Dollar pro Monat. Max kostet 100 bis 200 Dollar pro Monat mit mehr Kapazitaet fuer komplexe Projekte.

## Haeufige Fragen

**Welches Abo brauche ich fuer Claude Computer Use?**
Pro oder Max. Das Feature ist nicht im Free Plan enthalten.

**Funktioniert Computer Use auf Windows?**
Die Desktop App laeuft auf Windows. Die Bildschirmsteuerung ist derzeit nur auf macOS verfuegbar.

**Ist der Auto Mode sicher?**
Anthropic hat integrierte Safeguards eingebaut. Claude zeigt vor wichtigen Schritten seinen Plan. Du kannst jederzeit eingreifen. Trotzdem ist es ein Research Preview und sollte nicht fuer produktionskritische Aufgaben ohne Aufsicht genutzt werden.

**Kann ich alle 22 Features gleichzeitig nutzen?**
Ja. Die Features ergaenzen sich. Cowork ist die Plattform, Claude Code das Entwicklertool, Plugins erweitern die Funktionalitaet.
