Anthropic hat zwischen Januar und März 2026 22 bedeutende Updates für Claude veröffentlicht. Das sind im Schnitt fast acht pro Monat. Ein viraler X Post von TawohAwa fasst die Entwicklung zusammen. Hier der komplette Ueberblick über alle Features und was sie für Nutzer bedeuten.

## Alle 22 Updates chronologisch

### Januar 2026

Claude Cowork Launch. Die Desktop App die Claude zum Arbeitsassistenten macht.

### Februar 2026

Opus 4.6 Release mit verbessertem Reasoning und 1 Million Token Kontext in der Beta. Sonnet 4.6 Release als neüs Standard Modell. Cowork auf Windows und macOS. PowerPoint Integration für direkte Zusammenarbeit in Präsentationen. Excel Integration für automatisierte Tabellen und Analysen. Cowork Plugins für erweiterbare Skills. Claude Code Security mit Schwachstellen Scans. Claude Code Remote Control. Scheduled Tasks in Cowork für wiederkehrende Jobs. Connector im Free Plan für alle Nutzer kostenlos.

### März 2026

Claude Memory jetzt kostenlos für Free User. Claude Marketplace für fertige Skills und Vorlagen. Claude.com Ambassadors Community Programm. Code Review für Claude Code. Claude Skills für Excel und Slides. Charts und Diagramme direkt im Chat. 1 Million Token Kontextfenster. Dispatch für intelligente Task Verteilung. Claude Code Channels für Team Entwicklung. Cowork Projects für Projektverwaltung. Claude Computer Use für Desktop Steürung. Auto Mode in Claude Code: Claude trifft eigenständig Permission Entscheidungen mit integrierten Safeguards.

## Was die wichtigsten Updates in der Praxis bedeuten

### Cowork und Desktop Integration

Claude ist nicht mehr nur ein Chatbot. Cowork macht daraus einen Arbeitsassistenten der mit Excel, PowerPoint, Slack und anderen Tools verbunden ist. Teams die mit Microsoft Office arbeiten können Claude direkt in ihren bestehenden Workflow einbinden.

### Opus 4.6 und das 1 Million Token Fenster

Das 1 Million Token Kontextfenster ermöglicht es komplette Codebasen oder umfangreiche Dokumentensammlungen in einer einzigen Anfrage zu verarbeiten. Für Entwickler die grosse Projekte analysieren müssen ist das ein konkreter Produktivitätsgewinn.

### Auto Mode in Claude Code

Der am 24. März vorgestellte Auto Mode ist das jüngste und weitreichendste Update. Claude trifft eigenständig Entscheidungen über Dateiänderungen und Terminal Befehle. Sicherheits Checks laufen im Hintergrund. Entwickler müssen nicht mehr jede Aktion einzeln bestätigen.

### Computer Use und Dispatch

Claude kann jetzt den Mac Desktop steürn. Apps öffnen, im Browser navigieren, Tabellen ausfüllen. Dispatch ermöglicht es Aufgaben vom Smartphone aus zu starten. Du weist Claude eine Aufgabe zu und kommst später zu fertiger Arbeit zurück.

## Was kostenlos und was kostenpflichtig ist

Memory und Connector sind seit März 2026 im Free Plan enthalten. Cowork, Computer Use und Auto Mode erfordern ein Pro oder Max Abo. Pro kostet 20 Dollar pro Monat. Max kostet 100 bis 200 Dollar pro Monat mit mehr Kapazität für komplexe Projekte.

## Häufige Fragen

**Welches Abo brauche ich für Claude Computer Use?**
Pro oder Max. Das Feature ist nicht im Free Plan enthalten.

**Funktioniert Computer Use auf Windows?**
Die Desktop App läuft auf Windows. Die Bildschirmsteürung ist derzeit nur auf macOS verfügbar.

**Ist der Auto Mode sicher?**
Anthropic hat integrierte Safeguards eingebaut. Claude zeigt vor wichtigen Schritten seinen Plan. Du kannst jederzeit eingreifen. Trotzdem ist es ein Research Preview und sollte nicht für produktionskritische Aufgaben ohne Aufsicht genutzt werden.

**Kann ich alle 22 Features gleichzeitig nutzen?**
Ja. Die Features ergänzen sich. Cowork ist die Plattform, Claude Code das Entwicklertool, Plugins erweitern die Funktionalität.
